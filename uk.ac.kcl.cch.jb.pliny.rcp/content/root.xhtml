Welcome to Pliny

Welcome to Pliny, and Pliny's welcome screen. Pliny is probably rather unlike other pieces of software with which you are already familiar such as a text editor or word processor. Pliny has, therefore, several resources available to help you get started:

  • Help System: Pliny has an extensive set of help pages, including a set of starting tutorials, that will introduce you to how to use Pliny's consitituent parts. You can click here to start the help system, or you can use the Help menu: help/help contents.
  • Tutorials: Contained within Pliny's help pages are a set of tutorials. Click here to see them.
  • Pliny's website: The website for the Pliny project can be seen at http://pliny.cch.kcl.ac.uk.
  • Closing the Welcome Screen: To close this welcome screen, and start using Pliny, click here. If you wish to see the welcome screen again, choose welcome from the help menu.

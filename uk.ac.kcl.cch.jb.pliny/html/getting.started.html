1. Getting Started with Pliny

Getting started with Pliny involves understanding two things. First, since Pliny is an Eclipse application, one needs to understand how Eclipse organises and manages both its window; and within it Pliny's visual components. Then, one needs to understand how to use those Pliny components to record and work with your materials.

  • The section Pliny and Eclipse introduces the basic elements of the Eclipse user interface that applies to Pliny.
  • The section Pliny Tutorials introduces some basic techniques for using Pliny

If you are new both both Eclipse and Pliny at the same time, we suggest that you work through the first of the Pliny tutorials, and then return here to review the next section on Pliny and Eclipse.

Pliny works on the Macintosh, but it looks a little different from how it looks under Windows. There is a little about the differences at Pliny on the Macintosh.

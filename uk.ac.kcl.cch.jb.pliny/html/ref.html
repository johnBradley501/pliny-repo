2. Pliny Reference

Pliny has a number of objects that you can use to annotate materials, and to organise your notes:

  • The Resource Explorer provides a starting list of all materials you have collected in Pliny, including all digital objects you have annotated and all the notes you have created.
  • The Image Annotator provides a place where digital images can be annotated.
  • The PDF Annotator provides a place for annotating PDF (Acrobat) files.
  • The Note Editor provides a place to edit a note, and link it to other notes.
  • Pliny's Web Browser container displays a web browser to you and allows you to attach notes to any page it displays.
  • The Annotation View allows you to associated Pliny's notes with other Eclipse tools that are not aware of Pliny.

There are also several other views associated with Pliny that provide various kinds of support functions:

  • The Containment View provides a graphical visualisation to help you understand how references to Pliny resources are contained with each other.
  • The Note Search View allows you to search through the words you have put in your notes to find instances where the word you are asking for appears.
  • The Type Manager View allows you to manage Pliny's link types.
  • The Pliny Resource Holder can be helpful as a temporary holder of references to Pliny notes or to other resources when you want to copy them from one place to another.

Finally, there are a number of subcomponents which appear in several different contexts within the Pliny system:

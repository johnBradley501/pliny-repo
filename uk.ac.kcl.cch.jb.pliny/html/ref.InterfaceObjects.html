2.1 Annotation Tools and Views

Pliny has a set of components that support annotation, notetaking and organisation of notes. These tools divide into two broad groups (defined for Pliny by the Eclipse framework in which it is built):

  • Editors: In Pliny, all the eclipse editor objects are annotation tools. All active instances of these annotation tool/editors appear in the Eclipse editor pane which usually takes up a central area on the Pliny window. There can be only one editor pane. It is possible to install additional tools into Pliny (not described here), but "out of the box" Pliny's Editor objects are:
    • Pliny's Web Browser container displays a web browser and allows you to attach notes to any page it displays.
    • The Note Editor provides a place to edit a note, and link it to other notes.
    • The PDF Annotator provides a place for annotating PDF (Acrobat) files.
    • The Image Annotator provides a place where digital images can be annotated.
  • Views: Views provide tools to assist you in managing your use of Pliny -- think of them as supporting the name annotation, notetaking, note organising task that occurs with the annotation tools listed above. Views sit in one or more view panes (they cannot be mixed in with editors in the same pane). Although only one instance of a view can be open at a time, there can be more than one pane open with views in it. Often view panes are laid out around the central editor pane. For example, Pliny's Resource Explorer is a view and is (initially at least) placed to appear in a pane to the right of the main editor pane area. Pliny's containment view, however, often works best in a pane that is the full width of the Pliny/Eclipse window, and when opened is often placed in a pane across the bottom of the Pliny window. Pliny's views are:
    • The Resource Explorer provides a starting list of all materials you have collected in Pliny, including all digital objects you have annotated and all the notes you have created.
    • The Containment View provides a graphical visualisation to help you understand how references to Pliny resources are contained with each other.
    • The Note Search View allows you to search through the words you have put in your notes to find instances where the word you are asking for appears.
    • The Type Manager View allows you to manage Pliny's link types.
    • The Annotation View allows you to associated Pliny's notes with other Eclipse tools that are not aware of Pliny. This tool is only needed when Pliny is operating inside a larger Eclipe framework, rather than in the standalone packaging of Pliny, since all the annotators in Pliny's standalone packaging support integrated annotation already.
    • The Pliny Resource Holder can be helpful when you want to copy references to Pliny notes or other resources from one place to another.
